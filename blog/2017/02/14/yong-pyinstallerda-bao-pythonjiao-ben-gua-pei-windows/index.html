用Pyinstaller打包python脚本适配windows

Feb 14th, 2017 | Comments

一行命令的事，纯python脚本打包出来一般5Mb左右，大小还可以接收。

首先在脚本同目录下加入一个pyinstaller.py:

#!/usr/bin/env python
from PyInstaller.__main__ import run
run()

然后执行:

python pyinstaller.py -c -F <xxxx.py> -p <sdk>
  • <xxxx.py> 为脚本名称
  • 为三方包目录
