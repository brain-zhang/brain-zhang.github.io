以前一直认为C99 只支持const int 定义数组，今天才学到C99里面是支持动态数组的。


```
int main()
{
    int x = 12;
    char pz[x];
}

```
C99是支持的。在gcc4+以上的版本里测试OK。

另外stackoverflow上提到C++不支持动态数组，在g++4.4.7里面试了一下，也是可以的。

不是我不明白，这世界变化快。
